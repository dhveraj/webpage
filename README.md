# Palentrix Demo Product

High-conversion healthcare claims demo experience built with Next.js + Tailwind.

## Demo Flow
1. Login
2. Create patient
3. Generate ABHA ID (mock)
4. Upload discharge summary
5. AI extraction + editable review
6. Create claim draft
7. Review risk indicator and status
8. View dashboard and analytics

## Key UX Additions
- Guided 4-step claim wizard
- Demo Mode toggle (pre-filled patient + instant AI)
- Claim Risk Indicator (green/yellow/red + explicit reasons)
- Value-focused copywriting for hospital admins and insurance desks

## Routes
- `/` Login
- `/dashboard` KPI dashboard
- `/patients/new` Patient creation
- `/claims/new` Claim wizard
- `/claims/summary` Claim summary + risk
- `/analytics` Leadership analytics snapshot
